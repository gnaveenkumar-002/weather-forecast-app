# 🌦️ Weather Forecast Dashboard

A modern and responsive weather forecasting dashboard that provides real-time weather information, 5-day forecasts, location-based weather retrieval, and an intuitive user experience.

Designed and developed using modern frontend technologies with integration of the OpenWeather API to deliver accurate and up-to-date weather data.

---

## 🚀 Live Demo

🔗 Live Application: https://weather-forecast-app-8gth.vercel.app/

🔗 GitHub Repository: https://github.com/gnaveenkumar-002/weather-forecast-app

---

## 📌 Project Overview

The Weather Forecast Dashboard enables users to quickly access weather information for any city around the world. It combines real-time weather data, forecasting capabilities, Geolocation support, and responsive design into a clean dashboard interface.

The application focuses on usability, performance, and modern UI design while demonstrating practical frontend development skills and third-party API integration.

---

## ✨ Key Features

### 🌍 Weather Search

* Search weather conditions for any city worldwide.
* Displays accurate real-time weather information.

### 📍 Current Location Weather

* Uses the browser's Geolocation API.
* Retrieves weather information based on the user's current location.

### 🌡️ Real-Time Weather Details

* Current Temperature
* Weather Condition
* Humidity
* Wind Speed
* Feels Like Temperature
* Atmospheric Pressure
* Visibility

### 🌅 Sunrise & Sunset Information

* Displays local sunrise and sunset timings.

### 📅 5-Day Forecast

* Provides future weather predictions.
* Displays daily weather conditions and temperature trends.

### 🌙 Dark / Light Mode

* User-friendly theme switching.
* Theme preference stored using Local Storage.

### 🕒 Search History

* Maintains recently searched cities.
* One-click access to previous searches.

### ⚠️ Error Handling

* Handles invalid city names gracefully.
* User-friendly error messages.

### 📱 Responsive Design

* Optimized for:

  * Desktop
  * Laptop
  * Tablet
  * Mobile Devices
